This project implements deep-learning models to classify particle-collision events from the HLS4ML LHC dataset.

Project Overview:
Built multiple models including
Convolutional Neural Networks (CNNs) and
Autoencoder-based feature extractors.
Performed preprocessing, normalization, and dimensionality reduction for high-dimensional physics event data.
Benchmarked architectures for accuracy, latency, and generalization.
